مشروع براتشو كار - نسخة واجهة أمامية جاهزة للتعديل

المحتويات:
- index.html الصفحة الرئيسية
- pages/listings.html صفحة جميع الإعلانات
- pages/details.html صفحة تفاصيل السيارة
- pages/add-listing.html صفحة إضافة إعلان
- pages/chat.html صفحة الدردشة
- pages/account.html صفحة الحساب ولوحة التحكم
- assets/css/style.css التنسيق الكامل
- assets/js/app.js البيانات التجريبية والوظائف البسيطة
- manifest.json لتجربة PWA كتطبيق قابل للتثبيت

طريقة التشغيل:
1) فك الضغط.
2) افتح index.html مباشرة أو ارفع المشروع على GitHub Pages / Netlify / Firebase Hosting.
3) لتحويله إلى مشروع حقيقي:
   - اربطه مع Firebase Authentication
   - اربطه مع Firestore / Supabase / Laravel / Node.js
   - أضف رفع الصور حتى 20 صورة لكل إعلان
   - أضف بوابة دفع ليبية وخطط اشتراك شهرية/سنوية

ملاحظات:
- التصميم مستوحى من أسلوب منصات الإعلانات المبوبة لكن بهوية براتشو كار الخاصة.
- هذه نسخة Frontend كاملة كبداية قوية وسريعة، وليست Backend جاهز للإنتاج.
